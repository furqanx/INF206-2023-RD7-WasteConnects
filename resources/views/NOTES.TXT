Komponen Post dan Profile perlu ditinjau ulang 

resources
└── views
    ├── auth                  // folder untuk halaman autentikasi
    ├── components            // folder untuk menyimpan komponen 
    ├── layouts               // folder untuk layout aplikasi
    ├── livewire              /** sedang dalam pembuatan **/
    ├── pages                 // folder untuk menyimpan halaman


Contoh Format : 

├── product
    ├── create.blade.php  // halaman untuk menambahkan produk baru
    ├── edit.blade.php    // halaman untuk mengedit produk yang sudah ada
    ├── index.blade.php   // halaman untuk menampilkan daftar produk
    └── show.blade.php    // halaman untuk menampilkan detail produk


├── profile
    ├── edit.blade.php  // halaman untuk mengedit informasi profil
    └── show.blade.php  // halaman untuk menampilkan informasi profil
